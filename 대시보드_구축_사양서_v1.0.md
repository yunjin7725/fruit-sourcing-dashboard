# 🍎 과일 소싱 및 마진 분석 대시보드 구축 사양서 (Ver 2.0)

> **최종 업데이트:** 2026-03-26  
> **실제 구현 상태 기준으로 전면 갱신된 버전입니다.**

---

## 📌 프로젝트 개요

과일 셀러가 데이터를 기반으로 최적의 품목을 소싱하고, 마진 계산 및 경쟁사 분석을 통해 수익을 극대화할 수 있도록 돕는 **5페이지 통합 의사결정 웹 대시보드**.

- **기술 스택:** Python Flask (백엔드) + HTML/CSS/JS (프론트엔드)
- **외부 API:** KAMIS 농산물 유통정보 (도매가 조회)
- **데이터 소스:** KAMIS API + 3개 도매처 CSV 파일 (econfarm, mgb2bmall, hwanggs3)
- **실행 방식:** `python app.py` → `http://127.0.0.1:5000/` 로컬 웹서버 구동

## 🟢 Page 1: [Trend] 소싱 분석

**목적:** 현재 날짜 기준 계절별 소싱 타이밍 데이터와 AI 주간 가이드를 제공한다.

### 1.1 계절별 소싱 타이밍 (구현 완료)

- **기준 일자 선택:** 날짜 입력(date picker) → `조회` 버튼으로 갱신
- **백엔드 API:** `GET /api/season?date=YYYY-MM-DD`
- **3개 카드로 분류 출력:**

  | 카드 | 기준 | 설명 |
  |------|------|------|
  | 🚨 소싱 준비 (시즌 시작 2주 전) | 2주 후 시즌 진입 예정 품목 | 농가/도매처 컨택, 테스트 및 상품 페이지 세팅 필요 |
  | 🔥 현재 성수기 (판매 적기) | 현재 시즌 중인 품목 | 수요 최고, 마케팅비 집중 권장 |
  | ⚠️ 시즌 아웃 (종료 1주 전) | 2주 후 시즌 종료 예정 품목 | 수확 감소·단가 급등 예상, 재고 소진 권장 |

- **품목 범위:** 사과, 배, 복숭아, 포도, 감귤, 단감, 바나나, 키위, 파인애플, 오렌지, 토마토, 방울토마토, 수박, 참외, 딸기, 멜론, 레몬, 체리, 망고, 자몽, 아보카도 (총 21개 품목)

### 1.2 AI 주간 소싱 가이드 (Output)

- **대상 품목:** 성수기 + 소싱 준비 품목 전체 (자동 취합, 중복 제거)
- **출력 항목 (품목별):**
  - 소싱 상태 태그 (🔥 현재 성수기 / 🚨 소싱 준비)
  - **권장 판매 단위:** 23개 품목별 하드코딩된 최적 판매 중량 (예: 딸기 500g·750g, 수박 4kg·6kg)
  - **추천 검색 키워드/태그 전략:** `#품목명산지직송`, `#고당도품목명`, `#특품품목명`, `#가정용품목명` 조합 자동 생성

## 🔵 Page 2: [Margin] 마진 계산기

**목적:** 모든 비용을 고려해 손해 보지 않는 정확한 판매가를 계산한다.

### 2.1 레이아웃 구조 (2열 배치)

- **좌측 설정 패널** + **우측 차트 및 단가 시뮬레이션**

### 2.2 설정 패널 입력값 (좌측)

| 입력 항목 | 기본값 | 설명 |
|----------|--------|------|
| 과일 품목/품종 선택 | 드롭다운 | KAMIS 기반 품목 목록 자동 로드 (`/api/fruits`) |
| 비교 기준 일자 | 오늘 날짜 | KAMIS 가격 조회 기준일 |
| 판매 단위 변환 | 1kg | 선택 중량: 1·2·3·5·10kg |
| 플랫폼 선택 | 쿠팡 (10.8%) | 쿠팡/스마트스토어/당근비즈니스/토스비즈니스/G마켓/옥션/직접 입력 |
| 수수료율 (%) | 10.8 | 플랫폼 연동 자동 반영, 직접 수정 가능 |
| 포장비/기타 (원) | 1,500 | 수동 입력 |
| 배송비 (원) | 3,500 | 수동 입력 |
| 마케팅 비용 (%) | 5 | 판매가 대비 비율 |
| 목표 마진율 (%) | 10 | 원가 대비 마진율 |

### 2.3 설정 패널 버튼 (좌측 하단)

1. **`차트 갱신`** 버튼: KAMIS API 호출 → 3개년 가격 차트 및 단가 시뮬레이션 실시간 갱신
2. **`📁 나의 소싱 관리장으로 이동`** 버튼: 클릭 시 **5페이지(소싱 관리장) 탭으로 바로 이동** *(신규 추가)*
   - 초록 계열 테두리/배경, 호버 시 글로우 효과

### 2.4 우측 - 3개년 가격 트렌드 차트

- **백엔드 API:** `GET /api/price_trend?item_code=&kind_code=&category_code=&date=`
- **차트 타입:** Chart.js 라인 차트 (선택 중량 기준으로 단가 환산)
- **표시 연도:** 최근 3개년 (예: 2024·2025·2026)
- **기능:** 포인트 위 가격 레이블 표시, null 데이터 보간(선형 보간)

### 2.5 우측 - 최종 단가 시뮬레이션

#### 기준 도매 원가 선택

1. **KAMIS 평균가** (라디오 기본 선택): 1kg당 KAMIS 도매가 × 선택 중량
2. **직접 입력**: 총 원가 수동 입력
3. **제휴 도매처 단가 비교 테이블** (매칭 품목 있을 시):
   - 3개 도매처(econfarm=도매처1, mgb2bmall=도매처2, hwanggs3=도매처3) CSV 매칭
   - 1kg당 단가 기준 오름차순 정렬, ⭐ 최저가 표시
   - 클릭 시 해당 단가로 마진 즉시 재계산

#### 최적 판매가 + 마진 분석 바

- **공식:** `판매가 = (총원가 × (1 + 목표마진율)) / (1 − 수수료율 − 마케팅비율)`
- **결과 출력:**
  - 최적 판매가 (100원 단위 반올림)
  - 3단 마진 분석 바: 총 사입 원가 / 수수료+마케팅비 / 순수익 (목표마진)

### 2.6 과일 품목 드롭다운 구성

- KAMIS `fruit_price_comparison_detailed.csv` 기반 자동 로드
- **확장 품종:** 딸기(설향·킹스베리·죽향·금실), 사과(후지·홍로·아오리·감홍·부사), 배(신고·원황·화산), 포도(캠벨·샤인머스캣·거봉), 토마토(일반·방울·대추방울·스테비아), 감귤(하우스·노지·천혜향·한라봉)

## 🟠 Page 3: [Price] 도매처 비교

**목적:** 3대 도매처의 선택 품목 가격을 비교하여 최적 공급처를 결정한다.

### 3.1 연동 방식

- **2페이지에서 품목 선택 후 차트 갱신 시 자동 연동** (currentPcode 전역 공유)
- `데이터 새로고침` 버튼으로 수동 갱신 가능

### 3.2 도매처 비교 테이블

| 컬럼 | 내용 |
|------|------|
| 도매처 | 도매처 1·2·3 (클릭 시 해당 사이트 링크 이동) |
| kg당 단가 | 선택 품목 CSV 매칭 최저 1kg 단가, 선택 중량 환산 가격 |
| 최소 주문량(MOQ) | 해당 품목 매칭 상품명 및 총 상품 수 |
| 배송비/조건 | "업체 직접 확인" (도매처 직접 링크로 확인) |
| 최종 선택 점수 | 최저가: ★★★★★ (초록), 나머지: ★★★★☆ (노랑) |

### 3.3 도매처 정보 (실제 연결)

| 내부명 | 표시명 | 실제 사이트 |
|-------|-------|------------|
| econfarm | 도매처 1 | econfarm.adminplus.co.kr |
| mgb2bmall | 도매처 2 | mgb2bmall.com |
| hwanggs3 | 도매처 3 | hwanggs3.adminplus.co.kr |

### 3.4 AI 종합 추천

- 최저가 도매처 자동 하이라이트 + 마진율 달성 가능 여부 문구 출력
- 로그인 정보 안내 표시 (공통 ID/PW)

## 🟣 Page 4: [Benchmarking] 쿠팡 1위 분석

**목적:** 1위~10위 경쟁 판매자 전략을 벤치마킹하여 차별화된 상품 전략을 수립한다.

### 4.1 연동 방식

- **2페이지 마진 계산기에서 선택한 품목을 자동으로 반영** (드롭다운 현재 선택값 읽기)
- 4페이지 접속 시 현재 선택 품목명 + 시뮬레이션 중량 자동 표시

### 4.2 상위 판매자 벤치마킹 (Top 10)

- `경쟁사 벤치마킹 실행` 버튼 클릭 시 AI 시뮬레이션 데이터 생성 (약 0.8초 딜레이)
- **출력 형태:** 순위(1~10위), 상품 제목, 판매가, 누적 리뷰 수 및 평점
- 1~3위는 골드 배지, 4~10위는 사이언 배지

### 4.3 나만의 차별화 전략 수립 (AI)

- **차별화된 상품 제목 제안:** 브릭스·산지직송·당일수확 등 고전환 문구 조합 자동 생성
- **SEO 키워드 태그:** 품목별 맞춤 태그 6개 (검색량 수치 + 경쟁 강도 표시)
  - 저경쟁(초록), 보통(노랑), 높음(주황) 색상 코딩
  - 지원 품목: 사과·딸기·토마토·배·감귤·수박·참외 (항목별 맞춤 태그 내장)
- **리뷰 분석:** 경쟁사 공통 불만 포인트 + 차별화 솔루션 제안

### 4.4 소싱 확정 저장 버튼

- `⭐ 이 꿀마진 품목을 최종 확정하여 '5. 내 소싱 관리장'에 저장` 버튼
- 클릭 시 현재 2페이지 마진 계산 결과(원가, 추천 판매가, 선택 중량)를 **LocalStorage**에 저장
- 저장 즉시 5페이지 `renderSourcingManager()` 갱신

## 🟤 Page 5: [My List] 소싱 관리장

**목적:** 검증 완료된 최적 소싱 품목들을 한눈에 관리하고 실전 판매에 투입한다.

### 5.1 저장 방식

- **저장 위치:** 브라우저 LocalStorage (`premiumSourcingItems` 키)
- **저장 항목:** 품목명, 판매 중량(kg), 예상 원가(원), 추천 판매가(원), 저장일자

### 5.2 카드 레이아웃

품목별 카드 형태로 그리드 출력:

| 항목 | 설명 |
|------|------|
| 품목명 + 저장일 | 카드 상단 헤더 |
| 판매 단위 | 저장 시 선택한 kg 수 |
| 예상 원가 | 도매/직접입력 기준 원가 (노랑) |
| 추천 판매가 | 마진 계산기 역산 결과 (사이언) |
| 예상 마진율 | `(판매가-원가)/원가 × 100` (초록) |
| ❌ 버튼 | 개별 삭제 (LocalStorage 업데이트 후 즉시 갱신) |

- 저장 품목 없을 시: "아직 저장된 소싱 품목이 없습니다." 안내 메시지
- `저장 내역 전체 삭제` 버튼: 전체 초기화 (confirm 확인 팝업 후 실행)

### 5.3 접근 경로

- 상단 탭 `5. [My List] 소싱 관리장` 클릭
- **2페이지 마진 계산기 좌측 패널 `📁 나의 소싱 관리장으로 이동` 버튼** *(신규)*
- 4페이지 저장 버튼 클릭 후 자동 이동 가능

## 🛢 백엔드 API 명세 (Flask / app.py)

| Endpoint | Method | 설명 |
|----------|--------|------|
| `/` | GET | index.html 메인 페이지 렌더링 |
| `/api/fruits` | GET | 과일 품목 드롭다운 목록 반환 (KAMIS 기반 + 확장 품종) |

| `/api/b2b_prices` | GET | 3개 도매처 CSV 데이터 반환 |
| `/api/upload_b2b` | POST | CSV/엑셀 업로드 → 런타임 도매처 데이터 추가 |

## 🛠 시스템 공통 가이드라인

1. 모든 데이터는 시각화된 **대시보드 형태(차트, 테이블, 카드)**로 제공한다.
2. KAMIS 데이터 NULL 발생 시 선형 보간 → 5년 평균(평년) 순으로 폴백 처리한다.
3. 도매처 CSV 매칭은 과일 이름 동의어/별칭 매핑 + 품종 오매칭 방지 필터를 적용한다.
4. 저장 데이터는 **브라우저 LocalStorage** 에 보관 (서버 재시작과 무관하게 유지).
5. 탭 이동 시 각 페이지 관련 데이터 자동 갱신 (sourcing-manager → renderSourcingManager 등).
